# knowledge

知识沉淀层的第一版共享包。

当前职责：

- 定义规则、模板、案例、实验结果、剧本等核心类型
- 提供系统初始知识资产和站点画像示例
- 提供知识资产查询入口
- 为后续 GEO / SEO 专有智能的结构化存储打基础

后续会继续扩展：

- 规则 DSL
- 案例与实验模型
- 检索接口
- 版本化与权重更新机制

当前已包含的知识资产类型：

- `KnowledgeRule`
- `KnowledgeTemplate`
- `KnowledgeCase`
- `KnowledgeExperimentResult`
- `KnowledgePlaybook`
